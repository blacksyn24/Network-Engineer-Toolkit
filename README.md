# 🌐 Network Engineer Toolkit

## 🧠 Description

**Network Engineer Toolkit** est une application web tout-en-un conçue pour les étudiants en réseau et les ingénieurs souhaitant automatiser et simplifier les tâches liées à l’adressage IP, la configuration réseau et la conception d’infrastructures.

Cette plateforme regroupe plusieurs outils essentiels dans une seule interface moderne, permettant de gagner du temps et de mieux comprendre les concepts du réseau informatique (niveau CCNA).

---

## 🎯 Objectifs

* Simplifier le calcul de sous-réseaux (VLSM)
* Aider à la conception de réseaux multi-sites
* Générer automatiquement des configurations Cisco
* Fournir un outil pédagogique et professionnel

---

## ⚙️ Fonctionnalités

### 🧮 Calculateur d’adressage IP

* IPv4 (VLSM complet)
* IPv6 (subnetting /64)
* Conversion :

  * Décimal ↔ Binaire
  * CIDR ↔ Masque

---

### 🌍 Planificateur réseau

* Génération de plans d’adressage multi-sites
* Organisation hiérarchique (site → LAN)
* Allocation automatique des sous-réseaux

---

### 🌐 Générateur de topologie

* Création de schémas réseau dynamiques
* Ajout de routeurs, switches et PC
* Visualisation des connexions

---

### ⚡ Générateur de configuration Cisco

Compatible avec les équipements de **Cisco** :

* VLAN
* ACL
* DHCP
* Routage statique

---

### 🔧 Outils supplémentaires

* Simulateur DHCP
* Calculateur VLAN
* Générateur ACL
* Outils IPv6
