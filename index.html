Ouvrir sans chercher le fichier

Raccourcis pour lancer le site dans votre navigateur par défaut (une fois le dossier du projet ouvert dans Cursor / VS Code).

  • Cursor / VS Code : Ctrl+Shift+B lance la tâche de build par défaut → ouvre index.html.
  • Terminal (Linux / macOS) : depuis le dossier du projet, exécutez le script ci-dessous.
  • Windows : double-cliquez sur open-netkit.bat dans l’explorateur de fichiers.
  • Menu des tâches : Ctrl+Shift+P → tapez Tasks: Run Task → Ouvrir Network Engineer Toolkit.
